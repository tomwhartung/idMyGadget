IdMyGadget

Simple and Flexible Device Detection for LAMP Websites

Simple Device Detection

With a single interface, IdMyGadget can detect just the basics, or up to hundreds of capabilities (when configured properly).

Flexible Device Detection

Supports device detection using the following third party packages:

  • Mobile-Detect: https://github.com/serbanghita/Mobile-Detect/blob/master/README.md
  • detectmobilebrowsers.com: http://detectmobilebrowsers.com/
  • Tera-WURFL: wurfl.sourceforge.net/wurfl.php

These device detection tools offer a wide variety of information about the devices. Also, the developers of these tools release them under different licenses.

IdMyGadget offers a common interface to these tools for LAMP (Linux Apache MySql PHP) sites.

For more information see device_detectors/index.php.
